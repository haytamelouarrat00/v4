- Developed a closed-loop visual servoing framework using 3D Gaussian Splatting as a differentiable, photo-realistic rendering backend, enabling real-time 6-DOF camera control driven purely by image feedback.
- Implemented and benchmarked feature-based IBVS leveraging XFeat (a learned sparse feature extractor) and MoGe-2 (a transformer-based monocular depth estimator from Microsoft) to construct the IBVS interaction matrix and recover metric scene geometry without depth sensors.
- Designed a photometric (direct) VS controller based on dense pixel-level error minimization, exploring classical and learning-informed optimization strategies (ESM, Levenberg-Marquardt) for robust convergence in high-dimensional image spaces.
